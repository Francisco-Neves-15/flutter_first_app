# Install

## SDK Download

**Manual Installation**: https://docs.flutter.dev/install/manual

- The compressed file can be found at: `flutter_windows_3.44.2-stable.zip`

- Extract the "flutter" folder

- Paste this folder into "C:\" (or wherever you prefer)

- Add `flutter\bin` in `PATH`

- In the terminal where the `flutter/` folder was pasted

- Run: `flutter --version`

- This will start the initial preparation of Flutter for the device

- After this process, any project can be started
